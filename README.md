# PB and React Authentication Page

## Features
- Fast
- Lightweight
- Server-less deployment
- Simple

## Requirements
- node.js
- pocketbase (https://pocketbase.io/)

## Setup Pocketbase Server
1.) Launch Command Prompt on its own or through Visual Studio Code and go to the directory for Pocketbase Decompressed Zip file.

2.) Run ````pocketbase.exe```` from Command Prompt and setup an Admin profile on the IP http://127.0.0.1:8090/_/

3.) Create a record as the initial user to login with.

## How to start React-side

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.
